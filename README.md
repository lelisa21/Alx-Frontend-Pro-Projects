# ALX Frontend Pro Projects Monorepo

Monorepo combining projects from the ALX Professional Frontend Development program. 

Projects follow a progressive learning path, with several building on each other — for example, the **Listing App** series evolves from scaffolding to a deployed Airbnb clone, while the **Project** series advances through foundational concepts to full applications like CineSeek (a movie discovery app). 

Consolidated from 18+ individual repos for easier navigation and continuity.

**Tech Stack:** Next.js, TypeScript, Tailwind CSS, GraphQL, PWA, REST APIs
